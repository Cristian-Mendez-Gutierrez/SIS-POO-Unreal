# SIS-POO-Unreal

Este repositorio está dedicado a proyectos y ejercicios de **Programación Orientada a Objetos (POO)** desarrollados para el motor **Unreal Engine** o clases relacionadas.

## 📝 Descripción
El objetivo principal de este espacio es almacenar implementaciones de lógica de programación utilizando **C++**, enfocándose en los pilares de la POO dentro del entorno de desarrollo de Unreal.

## 📂 Contenido del Repositorio
El proyecto incluye las siguientes carpetas y módulos principales:

*   **Cuenta_Bancaria**: Implementación de lógica para el manejo de cuentas bancarias (ejercicio clásico de POO).
*   **Piedra_Papel_Tijera_V1_Secuencial**: Primera versión del juego "Piedra, Papel o Tijera" con un enfoque secuencial.
*   **Piedra_Papel_Tijera_VM**: Una versión o módulo adicional del juego "Piedra, Papel o Tijera".

## 🛠️ Tecnologías Utilizadas
*   **Lenguaje:** C++ (100% del código del repositorio).
*   **Entorno:** Unreal Engine.
